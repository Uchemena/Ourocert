# OUROCERT - Bulk Certificate Generator

## What This Is

OUROCERT is a **complete bulk certificate generator**. Create beautiful certificates and send them to hundreds of people at once.

### What You Can Do:

1. ✅ **Create certificate templates** (upload your own or design from scratch)
2. ✅ **Generate certificates in bulk** (upload a spreadsheet, generate hundreds at once)
3. ✅ **Track generation progress** (monitor status, download individual or all certificates)
4. ✅ **Customize emails** (personalize subject and message for each batch)
